## Coupon Project
## 需求分析
#### 1. 小程序
- 优惠券使用者使用小程序参加活动，领取优惠券，参看个人信息
- 医病机构核销员使用小程序核销优惠券

#### 2. 后台管理
- 超级管理员进行系统管理，添加业务管理员，授权
- 业务管理员进行机构管理（CRUD），为机构管理核销员(CRUD)
- 业务管理员进行活动管理（CRUD），为活动添加优惠券(CRUD)

## 系统设计
- Django + Django Restframework 做后台与接口开发
- simple ui 更改 Django Admin 样式
